# VSTAR Global Trading website

A pure static product showcase website for **Wuxi Weixing Trading Co., Ltd.** It is built with plain HTML, CSS, and JavaScript and is ready for GitHub Pages.

## Preview locally

Open `index.html` directly, or run a local static server:

```powershell
python -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy to GitHub Pages

1. Create a GitHub repository and push all files in this folder.
2. Open **Settings → Pages** in the repository.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`, then save.

The enquiry form is serverless: submitting it opens a pre-filled WhatsApp message to the primary contact at `+86 151 6150 1775`. An additional WhatsApp contact, `+86 188 6200 5872`, is shown in the contact section and footer.
